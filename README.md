# NOTA-DEVOPS

Projeto de exercício DevOps com Node.js

## Descrição

API REST simples para gestão de cursos e alunos.

## Tecnologias

- Node.js
- Express
- PostgreSQL

## Como executar

```bash
npm install
npm start
```

## Endpoints

- GET /cursos - Lista cursos
- GET /alunos - Lista alunos
- POST /cursos - Cria curso
- POST /alunos - Cria aluno
